# データベース設計

## products

商品本体の管理用テーブル。

```sql
create table products (
  id uuid primary key,
  name text not null check (length(trim(name)) > 0),
  description text not null default '',
  brand_id uuid references brands(id),
  regular_price integer not null check (regular_price >= 1),
  sale_price integer,
  status text not null default 'draft',
  primary_category_id uuid references categories(id),
  created_at timestamptz not null,
  updated_at timestamptz not null,
  constraint products_sale_price_positive
    check (sale_price is null or sale_price >= 1),
  constraint products_sale_price_less_than_regular
    check (sale_price is null or sale_price < regular_price),
  constraint products_status_valid
    check (status in ('draft', 'active', 'archived'))
);
```
