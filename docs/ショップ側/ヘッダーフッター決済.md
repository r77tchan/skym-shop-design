## テーマ

- テーマはlocalStorageに保存する。リロード時にチラつかせないこと。

## お気に入り、カート、決済

- お気に入り、カートはlocalstorageに保存する。
- ログイン機能は作らない。
- 決済には、Stripe Checkout hostedを使用する。
  - 配送先の入力をStripe側で行う。
  - 送料をStripe側で選択できるようにする。
    - クリックポスト185円
    - 宅急便コンパクト600円
    - レターパックプラス600円
  - 商品情報を画像付きでStripe側に表示する。
  - Stripeから戻った際の画面は別途必要。
    - 成功時は注文内容を表示する為、session_idが必要。
    - /checkout/success?session_id={CHECKOUT_SESSION_ID}
    - /checkout/cancel
