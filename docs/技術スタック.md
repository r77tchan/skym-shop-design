# 技術スタック

## 採用スタック

| 領域                 | 採用技術                     | 用途                                                                   |
| -------------------- | ---------------------------- | ---------------------------------------------------------------------- |
| アプリケーション基盤 | Next.js App Router           | cache components採用                                                   |
| 言語                 | TypeScript                   |                                                                        |
| ホスティング         | Vercel                       |                                                                        |
| スタイリング         | Tailwind CSS v4 + CSS変数    | デザイン値をCSS変数で管理し、ユーティリティクラスでUIを組む            |
| UIコンポーネント     | shadcn/ui + Radix UI         | アクセシビリティを担保したUI部品の土台にする                           |
| アイコン             | lucide-react                 |                                                                        |
| フォーム             | React Hook Form              |                                                                        |
| validation           | Zod                          |                                                                        |
| サーバー状態管理     | TanStack Query               |                                                                        |
| ローカルDB           | IndexedDB + Dexie.js         |                                                                        |
| データベース         | Supabase Database / Postgres |                                                                        |
| ORM / DBアクセス     | Prisma Client                | 型安全目的。Prisma Migrateは使用しない。                               |
| 認証                 | Supabase Auth + Google OAuth | 管理画面のGoogleログインに使用する                                     |
| 認可                 | Postgres Row Level Security  |                                                                        |
| 画像ストレージ       | Cloudflare R2                |                                                                        |
| 決済                 | Stripe Checkout hosted       |                                                                        |
| メール送信           | Resend                       |                                                                        |
| 外部通知             | Slack Incoming Webhooks      | 運用者へのslack通知                                                    |
| リアルタイム通知     | Supabase Realtime            | 管理画面で注文や管理ログの更新通知を受け取る                           |
| 通知UI               | Sonner                       | 管理画面内のtoast通知、ショップ側のカート追加時等に使用する            |
| 監視                 | Sentry                       |                                                                        |
| E2E/APIテスト        | Playwright                   | ショップ側E2E、管理側E2E、Route HandlersやWebhookのAPIテストに使用する |
| DBスキーマ管理       | Supabase CLI migrations      | Prisma Migrateは使わずにこちらを使用する                               |
